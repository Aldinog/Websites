Expert Advisor

Kita menjual Software Expert Advisor (EA), Dengan lock nama atau lock Akun Trading dengan bentuk file
Apa perbedaan lock nama dan lock akun trading?
Lock nomor akun adalah mengunci file EA agar hanya dapat di gunakan untuk 1 id akun yang sudah terdaftar di dalam file EA, jadi selain id akun yang sudah terdaftar tidak akan bisa menggunakan file EA tersebut.

Lock nama adalah mengunci file EA agar hanya dapat di gunakan untuk nama akun yang sudah terdaftar di dalam file EA, jadi selain nama akun yang sudah terdaftar tidak akan bisa menggunakan file EA tersebut.
Apa kelebihan dari Lock nama?
kelebihan dari lock nama adalah bisa digunakan untuk banyak akun dan di broker manapun yang penting nama akun sama dengan nama yang sudah di daftarkan.

Copy Trade

Men-Copy semua transaksi trading Master langsung ke akun Trading Anda, tanpa pasword
Apakah Copy Trade aman?
Ya... Copy Trade lebih aman, karena master trading tidak mengetahui pasword akun trading kita
Copy trade hanya menyalin transaksi dari akun master ke akun member, member tetap memiliki akses penuh ke akun tradingnya sendiri.
Broker akun trading apakah harus sama dengan broker akun master?
Tidak... Broker bisa berbeda, tetapi master akan merekomendasikan broker tententu untuk menghindari spread yang melebar
